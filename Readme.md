# Vue video call using twilio

## Client setup

```
cd app-client
```

```
npm install | yarn
```

### Compiles and hot-reloads for development
```
yarn run serve
```

## Server setup
```
cd server
```

```
npm install | yarn
```

### Compiles and hot-reloads for development
```
node server
```


# Try it
go to: ``http://localhost:8080`` and try it
